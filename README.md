### Вимоги

- Додаты `.env` файл, в нього додати `KEY` згідно `.env.example`

- Node.js версія вказана у файлі `.nvmrc`

Для встановлення потрібної версії Node.js:

```bash
nvm use
```

### Встановлення залежностей

```bash
npm install
```

### Запуск додатку

У першому терміналі запустіть json-server:

```bash
node index.js <privateKey> <amount> <recipient>"
```
